position:fixed und Anker - Beispieldokument zur alternativen JavaScript-Kombinationslösung

Anker A — Anker B — Anker C — Anker D — Anker E

Bitte wählen Sie einen Link zu einem der untenstehenden Anker an. Wenn JavaScript aktiviert ist, wird beim Klick auf das Dokument eine Funktion aufgerufen. Beim Klick auf einen dokumentinternen Link sollte der Browser den Anker zunächst so anspringen, dass er unterhalb des fest positionierten Kopfbereichs liegt. Die Funktion verschiebt daraufhin Fokus nach oben, sodass der Anker hervorkommt und sich die jeweiligen Überschrift an den unteren Rand des fest positionierten Kopfbereichs anschließt. Als Verzögerungszeit wurden 50 Millisekunden angegeben. Wenn das Dokument mit einem Anker aufgerufen wird, sollte der gewählte Anker beim Laden ebenfalls durch automatisches Blättern nach oben sichtbar gemacht werden.
